EE task manager
The EE manager allow to send request for computaion to GEE. In particular, for image collection export. Furthure more it include a confirm manager allows overwriting the normal confirm behavior. This is useful to use the ee.data.* function. In particular to automatically create / fill / delete collections.
Insert function in code editor
This allow to automatically insert a function signature from documentation to the code editor.
Check if scripts are shareable
This allows to automatically check if a script shared with "get link or "copy script path" is shareable. It checks resources and script path. Your scripts are not transferred, however a list of all assets contained in the script is transferred to an external server to check general availability.
